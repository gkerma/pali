# Oracle 48 cartes – WebUI Streamlit

Cette appli propose un oracle de 48 cartes :

- Tirage libre (1 à 5 cartes)
- Tirage en croix (5 cartes)
- Thème clair / sombre
- Historique des tirages (pendant la session)
- Effet flip 3D façon tarot (recto / verso en hover / touch)

## Installation

pip install -r requirements.txt
